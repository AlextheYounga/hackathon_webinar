# macOS

macOS is popular among developers because it has a Unix-like terminal while still being beginner-friendly.

## Pros

- Good terminal experience
- Works well with Git, Node.js, Python, and most developer tools
- Strong support for VS Code
- Homebrew makes installing developer tools easier
- Similar enough to Linux for many tutorials

## Cons

- Apple Silicon can occasionally cause package compatibility issues
- Some open-source tools assume Linux, not macOS
- System Python can be confusing
- Homebrew paths can confuse beginners
- Some tools require extra permissions

## Recommendation for Students

If you are using macOS:

- Install Homebrew
- Install Git
- Install Node.js LTS
- Use VS Code or Cursor
- Test your project before the hackathon
