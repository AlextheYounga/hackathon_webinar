# Operating Systems

## Why Operating Systems Matter

Some AI tools run entirely in the browser.

Other AI tools run on your actual computer and need access to:

- Your files
- Your terminal
- Git
- Node.js
- Python
- Databases
- Environment variables
- Package managers

This means the same AI tool may work differently on Windows, macOS, and Linux.

## The Big Idea

Browser-based tools avoid most setup problems.

Local coding agents are more powerful, but they depend on your machine being configured correctly.
