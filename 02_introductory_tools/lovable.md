# Lovable

Lovable is useful for turning a natural-language idea into a web app.

## Good For

* Full-stack web app prototypes
* Landing pages
* Internal tools
* Simple SaaS-style apps
* Nontechnical or semi-technical teams

## Watch Out For

* Do not assume the generated app is secure
* Do not assume the database is designed well
* Do not add too many features too quickly
* Make sure the demo path works

## Example Prompt

```md
Build a simple hackathon project called StudyBuddy.

It helps students create flashcards from class notes.

Requirements:

- Upload or paste notes
- Generate 10 flashcards
- Let the user mark cards as known or unknown
- Show a progress score
- Use a clean modern design
- No login required for the demo
```
