# [Git](https://git-scm.com)

Git tracks changes to your code over time.

**Why use it during a hackathon:**
- Save checkpoints so you can undo mistakes
- Collaborate without overwriting each other's work
- Deploy directly from a repository

**The bare minimum:**
- `git init` — start tracking a project
- `git add .` — stage your changes
- `git commit -m "message"` — save a checkpoint
- `git push` — upload to GitHub or another remote

**GUI clients (no terminal needed):**
- [GitHub Desktop](https://desktop.github.com) — simple, good for beginners
- [Sourcetree](https://www.sourcetreeapp.com) — more features, free
- [GitKraken](https://www.gitkraken.com) — polished, paid but free for students
- [VS Code](https://code.visualstudio.com) — has built-in Git UI (source control tab)
