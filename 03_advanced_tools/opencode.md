# [OpenCode](https://opencode.ai)

OpenCode is an open-source AI coding agent.

It is useful for students who want a terminal-first workflow and more control over model providers.

## Good For

* Developers who like the terminal
* Open-source workflows
* Trying different AI models
* Local coding workflows
* Advanced students who want control

## Watch Out For

* Setup can be more technical
* Provider configuration can be confusing
* API keys must be handled carefully
* Different models behave differently
* Terminal commands can change your machine

## Key Concept

OpenCode is not just a chatbot.

It is closer to an AI assistant that can participate in your development workflow.
