# [Cursor](https://cursor.sh)

Cursor is an AI-focused code editor.

It is useful because it feels familiar to students who already use [VS Code](https://code.visualstudio.com)-style editors.

## Good For

* Editing existing projects
* Asking questions about a codebase
* Inline code changes
* Autocomplete
* Agent-style coding tasks
* Students who prefer graphical editors over terminal tools

## Watch Out For

* It can still generate incorrect code
* Autocomplete can make bad assumptions
* You should review every change
* It is easy to accept code you do not understand
