# [OpenRouter](https://openrouter.ai)

OpenRouter is different from coding agents.

It is not mainly for editing your project.

It is an API layer for using many AI models through one interface.

## Good For

* Adding AI features to your own app
* Trying different models
* Comparing cost and quality
* Building chatbots
* Summarization
* Classification
* AI search
* Document analysis

## Example Use Cases

* A study app that summarizes notes
* A resume helper
* A chatbot for event attendees
* A code explanation tool
* A customer support bot
* A document question-answering app

## Watch Out For

* API keys cost money
* Models have rate limits
* Some models are better at coding, writing, vision, or reasoning
* Prompt quality matters
* Never expose API keys in frontend code
* Never commit API keys to [GitHub](https://github.com)
