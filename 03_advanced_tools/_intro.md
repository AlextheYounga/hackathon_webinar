# Advanced AI Tools

## Goal

This section is for students who are comfortable with code, terminals, Git, and local development.

Advanced AI tools can do much more than generate snippets.

They can:

- Read your project
- Edit multiple files
- Run commands
- Debug errors
- Write tests
- Refactor code
- Explain unfamiliar codebases
- Help with Git workflows

But they also require more responsibility.

---

# Local Coding Agents

Examples:

- Claude Code
- OpenCode
- Cursor

These tools work closer to your real development environment.

## Why They Are Powerful

They can work with the full context of your project.

Instead of asking:

```md
Write a React component.
````

You can ask:

```md
Read this repository.

Find the main app entry point.

Explain the structure.

Then propose the smallest change needed to add a dashboard page.

Do not edit files yet.
```

## The Most Important Rule

Do not let the agent make huge changes without a plan.

Good workflow:

1. Ask it to inspect
2. Ask it to explain
3. Ask it to propose a plan
4. Approve one small step
5. Run the project
6. Commit working changes
7. Repeat


# Advanced Tool Summary

Use advanced tools when:

* You understand the project structure
* You can use Git
* You can run the app locally
* You can read the code the AI writes
* You can recover if something breaks

Advanced tools are powerful because they can change real files.

That is also why they are dangerous.
